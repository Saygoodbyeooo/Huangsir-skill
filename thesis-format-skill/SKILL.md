# Skill: 江西财经大学现代经济管理学院本科毕业论文 DOCX 排版

## 触发条件

当用户要求生成、排版、格式化一份**江西财经大学现代经济管理学院本科毕业论文的 Word（.docx）文件**时触发本 Skill。

---

## 前提

- 使用 Python 3.8+ 和 `python-docx` 库（`pip install python-docx`）
- 所有格式参数定义在 `scripts/config.py`，不得在其他文件中硬编码
- 格式规范严格遵循《江西财经大学普通本科毕业论文指导手册》

---

## 执行流程

```
1. 收集用户信息：论文标题、姓名、学号、院系、专业、指导教师、日期
2. 询问用户是否已有章节内容（有 → 填入；无 → 生成占位骨架）
3. 调用 scripts/thesis_formatter.py 生成 .docx 文件
4. 提示用户：在 Word 中打开 → Ctrl+A 全选 → F9 更新所有域
5. 如需检查已有文档，调用 scripts/style_checker.py
```

## 格式规范

### 页面设置

| 项目 | 值 |
|------|-----|
| 纸张 | A4 |
| 上下页边距 | 2.54 cm |
| 左页边距 | 3.17 cm |
| 右页边距 | 3.17 cm |

### 页眉页脚

| 区间 | 页眉 | 页码 |
|------|------|------|
| 中文摘要 → 目录 | **无** | 罗马数字 Ⅰ Ⅱ Ⅲ，页脚居中 |
| 正文章节（第1章 → 最后一章） | "江西财经大学现代经济管理学院普通本科毕业论文"，五号宋体居中，下框线 | 阿拉伯数字，小五号 TNR，页脚居中 |
| 参考文献 / 附录 / 致谢 | **无页眉** | 阿拉伯数字续编，页脚居中 |

### 摘要

| 元素 | 字体 | 字号 | 说明 |
|------|------|------|------|
| "摘要" 标题 | 黑体 | 小三 (15pt) | 居中 |
| 中文摘要正文 | 楷体 | 小四 (12pt) | 固定行距 22pt |
| 【关键词】标签 | 黑体 | 小四 (12pt) | 加方括号 |
| 关键词内容 | 楷体 | 小四 (12pt) | 固定行距 22pt |
| "Abstract" 标题 | Times New Roman | 小三 (15pt) | 居中 |
| 英文摘要正文 | Times New Roman | 小四 (12pt) | 固定行距 22pt |
| 【Key words】标签 | Times New Roman | 小四 (12pt) | 加方括号 |

### 目录

| 元素 | 字体 | 字号 | 其他 |
|------|------|------|------|
| "目录" 标题 | 黑体 | 小三 (15pt) | 居中 |

| 二级、三级标题条目 | 宋体 | 小四 (12pt) | 不加粗 |
| 参考文献/附录/致谢条目 | 宋体 | 四号 (14pt) | 不加粗 |

| 元素 | 字体 | 字号 | 对齐 | 行距 | 段前/段后 |
|------|------|------|------|------|-----------|
| 论文题目 | 宋体 | 三号 (16pt) | 居中 | — | — |

## 文档结构（按顺序，分页符分隔）

1. 封面页
2. 中文摘要（含【关键词】）
3. 英文摘要（含【Key words】）— 分页符另起
4. 目录（TOC 域代码，dirty 标记强制 Word 打开时刷新）
5. 正文（新 section：页眉 + 阿拉伯数字页码）
6. 参考文献（新 section：**无页眉**，页码续编）
7. 附录（可选，无页眉）
8. 致谢（无页眉）

---

## 技术要求

1. 首行缩进必须使用 `w:firstLineChars`（字符单位），不用固定长度
2. 一级标题段前段后使用 `w:beforeLines` / `w:afterLines`（行单位），不用磅值
3. 标题使用倍数行距（防大字号裁切），正文使用固定行距
4. Heading 样式必须正确绑定到 python-docx 的 Heading 1/2/3，目录才能识别
5. TOC 域代码加 `w:dirty` 标记，Word 打开时自动提示更新
6. 参考文献/附录/致谢各自新建 section 以独立控制页眉
7. 页眉/页脚写入前彻底清空旧节点
